A Comparison of Clustering Methods

Three different algorithms were used to validate the clustering approach, each with unique strengths.

Method How it Works Key Strength
K-Means Partitions data into a pre-defined number of spherical clusters. Fast and efficient for large datasets.
Hierarchical Builds a tree of nested clusters, which can be cut at a desired level. Provides a rich, multi-level view of cluster relationships.
DBSCAN Groups together points that are closely packed, marking outliers as noise. Can find arbitrarily shaped clusters and is robust to outliers.
